# Sentinel Security Platform

Sentinel is the flagship cybersecurity product of Pinnacle Security, the company
founded by Grace Okoye. It is a threat-detection system that monitors networks for
intrusions and is designed to run efficiently on cloud infrastructure.

Okoye built Sentinel's core architecture on principles she developed while working on
hardware validation at Beacon Semiconductor under Leon Ashby. Her central insight was
that most security products monitored threats at the application layer while ignoring
patterns detectable at the network and hardware levels. Sentinel watches for anomalies
closer to the infrastructure, which means it catches classes of attacks that
application-layer tools miss.

Sentinel is offered as a managed service on Nimbus Systems' Stratus platform, through
a partnership that Pinnacle Security and Nimbus Systems established shortly after
Pinnacle's founding in 2012. The partnership makes Sentinel the default cybersecurity
option that Nimbus presents to its Stratus customers. Marcus Feld, who became Nimbus
Systems' CTO after the Quill acquisition, has endorsed Sentinel as the appropriate
security layer for companies running sensitive workloads on Stratus.

Beacon Semiconductor, whose Photon chips are used in Stratus data-center hardware,
benefits indirectly from Sentinel because the Pinnacle Security–Nimbus partnership
gives chip customers confidence that the infrastructure running those chips is well
protected.

Pinnacle Security has expanded Sentinel to cover on-premises environments as well as
cloud deployments, allowing enterprise customers to run a unified security posture
across both.
