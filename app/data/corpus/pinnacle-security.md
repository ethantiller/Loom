# Pinnacle Security

Pinnacle Security is a cybersecurity company founded in 2012 by Grace Okoye. Before
founding the company, Grace Okoye worked as an engineer at Beacon Semiconductor,
the chip company founded by Leon Ashby, where she developed an interest in securing
hardware and the software that runs on it.

Pinnacle Security's main product is Sentinel, a threat-detection system that
monitors networks for intrusions. Sentinel is designed to run efficiently on cloud
infrastructure.

Pinnacle Security partners with Nimbus Systems to protect the Stratus cloud
platform, offering Sentinel as an add-on service to Nimbus customers. This
partnership connects Pinnacle Security to the largest cloud provider in the
Elmsworth region.

Grace Okoye has spoken about how her years at Beacon Semiconductor shaped
Sentinel's low-level approach to security, watching for threats close to the
hardware rather than only at the application layer.
